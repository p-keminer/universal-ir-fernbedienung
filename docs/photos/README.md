# photos

## Zweck

Ausgewaehlte Fotos fuer Aufbau, Wiring und README.

## Regeln

- Nur repotaugliche Fotos einchecken.
- Private oder unsortierte Rohfotos nach `docs/photos/raw/`; dieser Ordner ist gitignoriert.
- Dateinamen mit Datum und Thema verwenden.

Beispiel:

```text
2026-05-01_breadboard_ir_driver.jpg
```
