# docs

## Zweck

Dieser Ordner enthaelt Arbeitsnachweise, Fotos, Messungen und Protokolle.

## Inhalt

- `templates/` fuer wiederverwendbare Protokollvorlagen
- `photos/` fuer ausgewaehlte Projektfotos
- `session_notes/` fuer kurze Arbeitsnotizen

## Regeln

- Dated Protokolle bevorzugen: `YYYY-MM-DD_thema.md`
- Reale Messwerte mit Aufbau, Versorgung und Beobachtung dokumentieren
- Fotos nur einchecken, wenn sie fuer GitHub und Nachvollziehbarkeit sinnvoll sind
- Private Rohdaten nach `docs/private/`, das per `.gitignore` ausgeschlossen ist
