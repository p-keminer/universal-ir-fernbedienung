# assembly

## Zweck

Dieser Ordner dokumentiert den mechanischen Aufbau.

## Erster Aufbau

Der erste sinnvolle Aufbau ist:

1. ESP32-S3 und IR-Treiber auf Breadboard
2. KY-005 so ausrichten, dass Testgeraete gezielt getroffen werden
3. LCD1602 lose anschliessen, sobald der IR-Pfad funktioniert
4. Bedienmodul erst danach mechanisch platzieren

## Spaeterer Loetstand

Fuer einen loetbaren Prototypen bieten sich an:

- kleine Lochrasterplatine fuer NPN-Treiber und Anschluesse
- Taster direkt auf Frontplatte oder eigener Button-Platine
- LCD1602 als sichtbares Frontmodul
- IR-LED vorne frei und stabil ausgerichtet
