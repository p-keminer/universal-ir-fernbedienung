# schematics

## Zweck

Dieser Ordner ist fuer Schaltplaene, Skizzen und Verdrahtungsbilder vorgesehen.

## Aktueller Stand

Noch kein finaler KiCad-Schaltplan vorhanden.
Die aktive Elektronikreferenz im Projektstil liegt in:

```text
../electronics/SCHEMATIC_CURRENT.md
```

Die v1-Gesamtskizze liegt in:

```text
IR_REMOTE_V1_ASCII.md
```

Der konkrete erste Breadboard-Schaltplan fuer den IR-Sendepfad liegt in:

```text
IR_TRANSMITTER_BENCH_STEP_01.md
```

## Format

Geeignet sind:

- Markdown-Skizze
- KiCad-Projekt
- exportiertes PNG/PDF
- Foto einer sauberen Handskizze

Rohdateien und Exporte sollten denselben Stand beschreiben.
