# wiring

## Zweck

Dieser Ordner enthaelt die reale Verdrahtungsdokumentation.

## Aktueller Stand

Noch kein bestaetigtes Wiring-Log vorhanden.
Fuer den ersten Aufbau die Vorlage nutzen:

```text
docs/templates/WIRING_LOG_TEMPLATE.md
```

## Muss dokumentiert werden

- ESP32-S3 Boardvariante
- IR-GPIO `GPIO4`
- Transistor-Typ und Pinout
- Basiswiderstand
- IR-LED-/KY-005-Vorwiderstand
- Versorgung des KY-005 oder der IR-LED
- gemeinsame Masse
- Foto des Minimalaufbaus

Nicht fuer v1 dokumentieren muessen:

- LCD1602
- Taster
- Joystick
